## Introduction

This app integrates Ariston Net into Homey to control the following device:

- Nuos.

## Usage

1. You must have a Homey Pro.
2. Install the [Ariston Homey app](https://homey.app/a/com.ariston) from the Homey App Store.
3. Pair your devices.
